# ⚡ Smart Energy Dashboard

نظام لوحة تحكم ذكي لإدارة ومراقبة استهلاك الطاقة في المباني، يتيح عرض البيانات بشكل تفاعلي عبر الأقسام (Sections) والغرف (Rooms).

## 🎯 مميزات المشروع
- عرض الاستهلاك الكلي للطاقة (Total View)
- إدارة الأقسام والغرف (إضافة / حذف / تعديل)
- عرض تفاصيل كل غرفة بشكل مستقل
- رسوم بيانية لعرض البيانات
- توصيات ذكية تعتمد على تحليل البيانات
- فلترة البيانات حسب التاريخ (يومي / أسبوعي / إجمالي)
- واجهة حديثة بتصميم داكن وألوان برتقالية

## 🛠️ التقنيات المستخدمة
- HTML
- CSS
- JavaScript
- Tailwind CSS
- Chart.js

## 📌 الهدف
تم تطوير هذا المشروع كنموذج واجهة (Frontend) لنظام إدارة طاقة ذكي يمكن ربطه لاحقًا بالذكاء الاصطناعي وواجهات خلفية (Backend).
